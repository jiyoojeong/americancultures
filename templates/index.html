How To Run

To Update AC Classes List
  1. Go to classes.berkeley.edu
  2. Mark the boxes for what semester you want for the courses (Ex: Fall 2020)
  3. Make sure the 'American Cultures' requirement portion is also marked
  4. Copy Paste the link into the box below.
  5. Click 'Update Courses'
